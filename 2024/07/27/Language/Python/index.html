基础语法

  1. 编码

    默认情况下，Python 3 源码文件以 UTF-8 编码，所有字符串都是 unicode 字符串。

  2. 标识符

    • 第一个字符必须是字母或者下划线_
    • 其他部分由数字、字母、下划线组成
    • 大小写敏感

  3. python保留字

  4. 几个陌生的保留字

    • from：：从模块中导入特定的函数或类

    • as：导入模块或重命名模块

    • assert：断言语句，用于测试条件是否为真

    • del：删除对象

    • try：尝试执行代码块，并捕获异常

    • except：异常处理，捕获并处理try块中的异常

    • finally：无论是否发生异常都会执行的代码块

    • raise：抛出一个异常

    • pass：空操作，不做任何事情

    • global：声明变量为全局变量

    • nonlocal：声明变量为非局部变量，用于闭包或嵌套函数

    • in：成员运算符，检查一个值是否存在于一个序列中

    • is：身份运算符，检查两个引用是否指向同一个对象

    • with：上下文管理器，用于简化资源管理

    • lambda：定义匿名函数

    • yield：在函数中使用，将函数转化为生成器

  5. 行与缩进

    python具有严格的缩进要求，用于取代{ }的功能

  6. 多行语句

    \实现多行语句

    在[]、{}、()，不需要使用\来定义多行语句

  7. 数字类型

    • 整数int：默认长整
    • 布尔bool
    • 浮点数float：1.23、3E-2
    • 复数complex：1+2j

  8. 字符串

    • python中单引号''和双引号""的使用完全相同

    • 使用三引号'''或"""可以指定一个多行字符串

    • 转义符\，可以使用r让\不发生转义，r"his is a line with \n"，则\会输出
    • +连接，*重复
    • 索引方式，从左往右0开始，从右往左-1开始
    • python中的字符串不能改变
    • 切片str[start:end]，前闭后开
    • 按步长切片str[start : end : step]

  9. 空行

    函数之间、类之间用空格一行分开，便于维护

  10. 一行之间多条语句

    ;隔开

  11. print输出

    默认换行输出，如果不想换行，结尾加上end=""

  12. import 与 from … import

  • 导入整个模块 import somemodule
  • 从某个模块导入某个函数 from somemodule import somefunction
  • 从某个模块导入多个函数 from somemodule import func1, func2, func3
  • 从某个模块导入全部函数 from somemodule import *

  4. 位运算符

    a = 60, b = 13

    | 运算符 | 描述 | 实例 |
    | ——— | —————————————————————————————— | —————————————————————————————— |
    | & | 按位与运算符：参与运算的两个值,如果两个相应位都为1,则该位的结果为1,否则为0 | (a & b) 输出结果 12 ，二进制解释： 0000 1100 |
    | | | 按位或运算符：只要对应的二个二进位有一个为1时，结果位就为1。 | (a | b) 输出结果 61 ，二进制解释： 0011 1101 |
    | ^ | 按位异或运算符：当两对应的二进位相异时，结果为1 | (a ^ b) 输出结果 49 ，二进制解释： 0011 0001 |
    | ~ | 按位取反运算符：对数据的每个二进制位取反,即把1变为0,把0变为1。~x 类似于 -x-1 | (~a ) 输出结果 -61 ，二进制解释： 1100 0011， 在一个有符号二进制数的补码形式。 |
    | << | 左移动运算符：运算数的各二进位全部左移若干位，由”<<”右边的数指定移动的位数，高位丢弃，低位补0。 | a << 2 输出结果 240 ，二进制解释： 1111 0000 |
    | >> | 右移动运算符：把”>>”左边的运算数的各二进位全部右移若干位，”>>”右边的数指定移动的位数 | a >> 2 输出结果 15 ，二进制解释： 0000 1111 |

    | not | not x | 布尔”非” - 如果 x 为 True，返回 False 。如果 x 为 False，它返回 True。 | not(a and b) 返回 False |

迭代器与生成器

迭代器

  • 迭代是 Python 最强大的功能之一，是访问集合元素的一种方式。
  • 迭代器是一个可以记住遍历的位置的对象。
  • 迭代器对象从集合的第一个元素开始访问，直到所有的元素被访问完结束，迭代器只能往前不会后退。

迭代器的两个基本方法：iter() next()

字符串、列表或元组对象都可用于创建迭代器

迭代器对象可以使用for语句进行遍历

也可以使用next函数

创建自定义迭代器

把一个类作为一个迭代器，需要在类中实现两个方法__iter__()与__next__()

  • __iter__()方法返回一个特殊的迭代器对象，这个迭代器对象实现了__next__()方法并通过StopIteration异常标识迭代的完成
  • __next__()方法会返回下一个迭代器对象

例子：创建一个返回数字的迭代器，初始值是1，逐步+1

StopIteration

StopIteration异常用于标识迭代的完成，以防止出现无限循环的情况，在__next__()方法中可以设置在完成指定循环次数后出发StopIteration异常来结束迭代

生成器

使用yield的函数被称为生成器，生成器函数是一种特殊的函数，可以在迭代过程中逐步产生值，而不是一次性返回所有结果。

跟普通函数不同的是，生成器是一个迭代器的函数，只能用于迭代操作

在生成器函数中使用yield语句，函数执行暂停，并返回yield后的表达式的值

每次调用生成器的next()方法或使用for循环进行迭代时，函数会从上次暂停的地方继续执行，知道再次遇到yield语句

函数

  • 语法

    不带return的函数体，默认返回None

  • 参数传递

    可更改和不可更改对象

    在py中，strings、tuples、numbers是不可更改对象，而list和dict是可更改对象

    • 不可变类型：变量赋值a=5后再赋值a=10，这里实际上是生成了一个新的int值对象，然后再让a指向它，而5被丢弃，而不是改变a的值，相当于新生成了a
    • 可变类型：变量赋值la=[1,2,3,4]后再赋值la[2]=5相当于对list的第三个元素修改，本身la没动，只是其内部的值修改了

    python函数的参数传递

    • 不可变类型：类似C++的值传递，如整数、字符串、元组。如fun(a)，传递的只是a的值，并没有影响a对象本身，如果在fun(a)内部修改a的值，则是生成一个a的对象

      可以看见在调用函数前后，形参和实参指向的是同一个对象（对象 id 相同），在函数内部修改形参后，形参指向的是不同的 id。

    • 可变类型：类似C++的引用传递，fun(la)内部修改la后，fun的外部la也受影响

  • 参数

    • 必需参数：必需以正确的顺序传入函数，调用时的数量和声明的时候一致

    • 关键字参数：关键字参数允许调用的时候顺序和声明时不一致，只需要写好关键字就可

    • 默认参数

      调用函数时，如果没有传递参数，会使用默认参数

    • 不定长参数：需要一个函数处理比声明时更多的参数，声明时不会命名

      加了星号 * 的参数会以元组(tuple)的形式导入，存放所有未命名的变量参数

      加了双星号**的参数会以字典（dict）的形式导入

      f.tell() 用于返回文件当前的读/写位置（即文件指针的位置）。文件指针表示从文件开头开始的字节数偏移量。f.tell() 返回一个整数，表示文件指针的当前位置。

    • f.seek()

      如果要改变文件指针当前的位置, 可以使用 f.seek(offset, from_what)函数。

      f.seek(offset, whence) 用于移动文件指针到指定位置。

      offset 表示相对于 whence参数的偏移量，from_what 的值, 如果是 0表示开头, 如果是 1 表示当前位置, 2 表示文件的结尾，例如：

      • seek(x,0)： 从起始位置即文件首行首字符开始移动 x 个字符
      • seek(x,1) ： 表示从当前位置往后移动x个字符
      • seek(-x,2)：表示从文件的结尾往前移动x个字符
